# Supraveghetor-Rutier

Sistemul monitorizează pulsul printr-un senzor biometric MAX30100 plasat la nivelul piciorului, detectând semnele de oboseală biologică înainte ca șoferul să adoarmă efectiv. Un fotorezistor LDR măsoară luminozitatea ambientală și ajustează automat frecvența testelor de atenție — mai des noaptea, mai rar ziua. Când sistemul detectează un potențial pericol, activează un buzzer sonor, iar șoferul trebuie să apese un buton fizic pentru a confirma că este conștient — un mecanism de tip „Dead Man's Switch". Un ecran LCD afișează în timp real starea sistemului, modul de funcționare și valorile pulsului. Spre deosebire de gadgeturile comerciale simple, care reacționează abia după ce șoferul a pierdut controlul fizic, acest prototip adoptă o abordare proactivă și adaptivă
